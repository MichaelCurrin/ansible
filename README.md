## Installation

### Install system dependencies

- Debian/Ubuntu
    ```sh
    $ sudo apt install -y ansible git
    ```


## Usage

```sh
$ sudo ansible-pull -U https://github.com/MichaelCurrin/ansible-playbooks.git
```
